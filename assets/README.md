# Assets Folder

Place your images in the following structure:

## Folder Structure

```
assets/
  images/
    logos/
      spotcarz_logo.png
      camera_icon.png
      book_icon.png
      eye_icon.png
    cards/
      card_bg_1.jpg
      card_bg_2.jpg
      card_bg_3.jpg
```

## Required Images

### Logos (in `assets/images/logos/`):
- `spotcarz_logo.png` - Main SpotCarz logo (used at the top of home page)
- `camera_icon.png` - Camera icon for "Spot Cars" card
- `book_icon.png` - Book icon for "Your collection" card
- `eye_icon.png` - Eye icon for "Share your collection" card

### Card Backgrounds (in `assets/images/cards/`):
- `card_bg_1.jpg` - Background image for "Spot Cars" card
- `card_bg_2.jpg` - Background image for "Your collection" card
- `card_bg_3.jpg` - Background image for "Share your collection" card

## Notes

- All images should be optimized for mobile devices
- PNG format is recommended for logos and icons
- JPG format is recommended for background images
- Ensure images are high resolution (2x or 3x) for better quality on high-DPI screens
